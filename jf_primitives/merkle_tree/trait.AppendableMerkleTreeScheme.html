Expand description

Merkle tree that allows insertion at back. Abstracted as a commitment for append-only vector.

Required Methods§

source

fn push( &mut self, elem: impl Borrow<Self::Element> ) -> Result<(), PrimitivesError>

Insert a new value at the leftmost available slot

  • elem - element to insert in the tree
  • returns - Ok(()) if successful

Provided Methods§

source

fn extend( &mut self, elems: impl IntoIterator<Item = impl Borrow<Self::Element>> ) -> Result<(), PrimitivesError>

Insert a list of new values at the leftmost available slots

  • elems - elements to insert
  • returns - Ok(()) if successful. If there are too many elements, insertions will be performed until the merkle tree is full, and wil return an Err().
