Module jf_primitives::signatures::bls_over_bn254

source ·
Expand description

This module implements the BLS signature over the BN254 curve. The BLS signature scheme relies on a bilinear map e:G1 x G2 -> GT and a hash function H from a message input space (e.g. bytes) to a group element G1. H must be such that for all m, the discrete logarithm H(m) w.r.t to the generator of G1 is unknown.

The scheme works as follows: Let g1 and g2 be generators of G1 and G2.

KeyGen()

  • sample a random s in the scalar field Fr and return the key pair (sk,pk):=(s,g2^s)

Sign(sk,m)

  • return sigma=H(m)^{sk}

Verify(pk,m,sigma)

  • Check that e(sigma,g_2)=e(H(m),pk)

In this module:

  • e is the pairing over the curve BN254 supported by the EVM EIP-196, EIP197
  • H is implemented using the “hash-and-pray” approach. See function hash_to_curve
