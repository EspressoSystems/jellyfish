    &mut self,
    elem: CommittedElemVar,
    path_vars: &MerklePathVar
) -> Result<Variable, CircuitError>

Computes the merkle root based on some element placed at a leaf and a merkle path.

  • elem - variables corresponding to the uid and the element value (e.g.: record commitment).
  • path_vars - variables corresponding to the Merkle path.
  • return - variable corresponding to the root value of the Merkle tree.
