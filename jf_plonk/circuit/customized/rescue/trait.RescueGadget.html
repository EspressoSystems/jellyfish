fn rescue_permutation(
    &mut self,
    input_var: RescueStateVar
) -> Result<RescueStateVar, PlonkError>

Given an input state st_0 and an output state st_1, ensure that st_1 = rescue_permutation(st_0) where rescue_permutation is the instance of the Rescue permutation defined by its respective constants

  • input_var - variables corresponding to the input state
  • returns - variables corresponding to the output state

Sponge-based hashes from Rescue permutations

  • data_vars - sponge input variables, data_vars.len() should be a positive integer that is a multiple of the sponge rate (i.e. 3)
  • num_output - number of output variables
