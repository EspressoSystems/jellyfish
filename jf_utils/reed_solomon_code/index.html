reed_solomon_erasure_decode
Decode into data_size data elements via polynomial interpolation. The degree of the interpolated polynomial is data_size - 1. First part of the share is the evaluation point, second part is its evaluation. Returns a data vector of length data_size. Time complexity of O(n^2).
